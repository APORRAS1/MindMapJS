Class: Globo

MM. Globo

Render de Globo. Se trata de un nodo con un texto y dentro de un globo. El render crea un grupo donde incluye los distintos elementos gráficos. Hereda del MM.NodoSimple.

new Globo(render, arbol, propiedades)

Parameters:
Name Type Description
render MM.Render Escenario donde pintar el mensaje
arbol MM.Arbol Capa donde pintar el grid
propiedades Object Conjunto de propiedades a establecer al nodo. Como Color fuente, posición, etc. Ver refencias Kinetic.Text, Kinetic.Blob, Kinetic.Line, Kinectic.Group.
